Overview

ChatKitty provides a Platform API, and a number of client SDKs to directly interact with the different types of resources representing data in your ChatKitty application. The ChatKitty Platform API is RESTful, using the HTTP protocol to expose discoverable HAL resources. The client libraries provide an asynchronous WebSocket based interface to ChatKitty's user-side functionality. While the client SDKs handle the requests and responses at the client-side, the Platform API provides an interface for you to manage and control your application server-side.

Note: The Platform API is not designed for client-side use.
